rem

rem用作字体、尺寸单位，1rem=浏览器默认font-size字体大小,即html根元素的font-size属性大小，大多数浏览器默认字体大小是16px。

em也是字体单位，但是1em相对大小是根据直接父元素的字体大小决定；
